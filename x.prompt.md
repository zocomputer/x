Use this prompt to post tweets to X (Twitter).

## Workflow

Use the X CLI to post a tweet. Assume the `x` CLI is already installed, follow the instructions in the repo README if not already installed. 

`x post "Your tweet text here"`

Repo for the X CLI and this prompt: https://github.com/zocomputer/x
